## 100 Day Roadmap

**Days 01–20 — Data Science Foundations**
- Missing value handling and data quality
- Exploratory data analysis
- Data visualisation with Matplotlib and Seaborn
- Feature engineering and encoding
- SQL for data science
- Statistics and hypothesis testing
- A/B testing analysis

**Days 21–50 — Machine Learning Core**
- Linear and logistic regression
- Decision trees and random forests
- XGBoost and gradient boosting
- Model evaluation metrics
- Cross validation and hyperparameter tuning
- Bias variance tradeoff
- Pipeline building with Scikit-learn

**Days 51–80 — Advanced Data Science**
- End-to-end ML pipelines
- NLP fundamentals and text classification
- Time series analysis and forecasting
- Clustering and unsupervised learning
- Dimensionality reduction with PCA
- Model interpretability with SHAP

**Days 81–100 — AI Engineering**
- LLM fundamentals and prompt engineering
- Embeddings and vector databases
- RAG system architecture
- FastAPI for ML model serving
- Model deployment on Render

## Daily Notebook Structure

Every notebook follows this exact structure:
```
## Problem Statement
What real-world question am I exploring today?

## Dataset
Where the data comes from, what it contains, and why I chose it.

## Analysis and Code
Full working code with markdown explanations throughout.

## Results
Charts, comparisons, and key findings with numbers.

## Key Learnings
3 things I discovered or understood better today.
```
